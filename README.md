# 📘 高中英語 6000 單字

一個**離線可用、免安裝、零追蹤**的高中英語單字學習 PWA（漸進式網頁應用程式）。
依**大考中心《高中英文參考詞彙表》（111 學年度起適用）6 級共約 6000 詞條**建置，用「間隔複習 ＋ 單元闖關 ＋ 錯題本」幫學生把學測必備單字背熟。

> 純單一 HTML 檔、資料全存在手機本機、不上傳雲端、不需註冊、沒有廣告。

> **版本**：v1.1（2026-07-04）　｜　🔵 圖示「ABC 6000」（藍）　｜　📱 響應式　｜　✍️ 填空題　｜　🔊 翻卡朗讀例句　｜　📝 學測考古盤點
> 字表來源：大學入學考試中心《高中英文參考詞彙表》（依 108 課綱編修，111 學年度起適用）

## ✨ 功能

- **6005 官方詞條**：依大考中心 6 級分級（第 1 級最易 → 第 6 級最難），每字含詞性、繁中字義、例句與中譯
- **6 級分級**：牌組可只選某一級（第 1～6 級）或全部；第 1～2 級約當國中程度、第 3～4 級為學測常考、第 5～6 級為加深加廣
- **單字卡**：3D 翻卡、滑動評分、翻面自動朗讀「單字＋例句」（TTS）、番茄鐘專注計時、SRS 間隔複習（Leitner 盒）
- **單元闖關**：分 121 單元（每單元 50 字，依級別排序），學習 → 測驗驗收 → ≥80% 過關蓋 ⭐ 解鎖下一關（可開自由模式跳關）
- **測驗**：英→中／中→英／拼字輸入／**填空**（例句挖空＋中文提示）四種題型、可選 10／20／50 題、可指定單元、中途存檔續做
- **錯題本**：答錯自動收錄、英中雙向各連對 2 次才畢業、間隔複習、錯因標籤、👹 魔王字、上次選錯提示
- **遊戲化**：XP 等級、連勝 combo、音效震動、多種徽章
- **今日任務**：學新字／複習／清錯題三段進度，綁連續天數
- **響應式版面**：手機直式單欄；平板 / 大螢幕自動放寬、單元多欄、卡片放大
- **📝 學測考古單字盤點**：過去 6 年（110–115）學測**整份考卷**出現、且在課綱 6000 字內的 **2570 字**（含克漏字、閱讀全文），視覺化卡片一點劃掉「已認識」，剩下不熟的一鍵變題庫直接背卡／測驗；可依**年份／級別**篩選，⭐標詞彙題正解、色條標級別、標籤標出現年數
- **學測倒數**：設考試日，自動算「每天該背幾字」
- **班級用**：學生匯出進度 JSON，老師用 `teacher.html` 拖入彙整成班級總表＋CSV
- **貼心設定**：深色模式跟系統、簡潔模式、減壓／家長模式、螢幕時間提醒、家長 PIN 鎖、聽覺自動發音、新手引導

## 🚀 使用方式

### 直接用
用瀏覽器開啟 `index.html` 即可。

### 安裝成 App（需用 GitHub Pages 等 https 網址開啟）
- **Android**：Chrome 開啟 → 選單 →「安裝應用程式 / 加到主畫面」
- **iPhone / iPad**：Safari 開啟 → 分享 →「加入主畫面」

安裝後可全螢幕、離線使用。

## 📁 檔案結構

| 檔案 | 說明 |
|---|---|
| `index.html` | 主程式（學生背單字）— 單檔即整個 App |
| `teacher.html` | 老師班級彙整頁（拖入學生匯出檔 → 總表 + CSV） |
| `manifest.json` / `sw.js` | PWA 設定與離線快取 Service Worker |
| `icon-*.png` / `apple-touch-icon.png` / `icon.svg` | App 圖示（Android / iOS） |
| `files/` | 官方字表與重建工具（`wordlist.json`、`words_full.json`、`content_progress.json`、`build_app.py`、`merge_content.py`、`smoke_test.py`、原始 `vocab.pdf`） |
| `README.md` / `更新記錄.md` | 說明與版本異動記錄 |

> 實際運作只需要 `index.html`、`teacher.html`、`manifest.json`、`sw.js` 與 icon 檔；`files/` 為資料／重建工具，不影響執行。

### 重建字表（開發用）
1. 補充內容寫入 `files/content_progress.json`（key＝小寫單字 → `{zh_tw, ex, exZh}`）。
2. 執行 `python files/build_app.py` 由 `words_full.json` ＋ 進度檔重建 `index.html` 的 `WORDS`。
3. 執行 `python files/smoke_test.py` 跑功能煙霧測試。

## 🔒 隱私

- 完全離線，**不連任何外部伺服器**（CSP `connect-src 'none'` 強制）。

- 無帳號、無追蹤、無廣告、無第三方程式碼。適合未成年使用。
